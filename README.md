# My First Cybersecurity Project

## Overview
This repository contains beginner-level cybersecurity tools built in Python.
The purpose of this project is to practice scripting, encryption concepts,
network scanning, and password strength validation.

│
└── README.md

## Tools Included

### 1. Caesar Cipher
A simple encryption and decryption script using substitution cipher logic.

### 2. Password Checker
Validates password strength based on length and character complexity.

### 3. Port Scanner
Scans target ports to check for open network services.
